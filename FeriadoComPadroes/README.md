
Desenvolver um sistema simples de vídeos (como streaming). O sistema deve reproduzir filmes e séries, controlar acesso de usuários, permitir adicionar funcionalidades (legenda, HD), integrar com um player externo e ser de fácil de uso.

Para isso, aplicamos os seguintes padrões de projeto:

Singleton (Garantir uma única instância da configuração)
Factory (Centralizar a criação de objetos)
Proxy (Criar um intermediário que controla o acesso)
Adapter (Criar um adaptador para um player externo)
Facade (Criar uma interface simples)
Decorator (Adicionar funcionalidades dinamicamente)
